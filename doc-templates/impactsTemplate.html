Types of changes / impacts

at-diff produces a list of changes and a list of impacts. This page lists the possible types of changes and impacts produced by at-diff.

Note that you should not rely only on at-diff to find possible breaking changes. at-diff does not replace tests. It cannot detect all breaking changes.
