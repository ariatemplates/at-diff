What

at-diff is a tool to compare different versions of a source code which uses the Aria Templates framework.

Unlike the traditional diff utility which mainly compares lines without interpreting them, at-diff mainly compares class methods or bean definitions and takes into account inheritance.

Why

By listing possible impacts, at-diff aims at detecting breaking changes, and at helping a developer to update some source code when one of the libraries it uses changed.

How

at-diff parses your code and/or the code of the libraries you use in your code. It analyses the differences between two versions and generates an html report with the list of changes and their impacts.

Please refer to the operations page to learn more about at-diff.
